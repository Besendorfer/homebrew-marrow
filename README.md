# homebrew-marrow

Homebrew tap for [**Marrow**](https://github.com/Besendorfer/marrow) — surface
the parts of a GitHub PR that matter, in your terminal.

## Install

```sh
brew tap besendorfer/marrow
brew install marrow
```

This installs the `marrow` CLI. (`brew install besendorfer/marrow/marrow` also
works without tapping first.)

## How the formula is maintained

`Formula/marrow.rb` is a prebuilt-binary formula generated by Marrow's
[`cli-release.yml`](https://github.com/Besendorfer/marrow/blob/main/.github/workflows/cli-release.yml)
workflow on each `cli-v*` release and attached to the GitHub release. After
cutting a release, drop that release's `marrow.rb` into `Formula/` here and push.

See [PUBLISHING.md](https://github.com/Besendorfer/marrow/blob/main/PUBLISHING.md)
in the main repo for the full release process.
